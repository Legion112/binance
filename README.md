Binance 
==================
Установка
===============================

Скачайте репозиторий

Выполнить команду:

```
composer install
```

Запуск тестового сервера
===============================
Чтобы запустить сервер выполните команду
```
php yii serve
```
После этого сайт будет доступен через  

http://localhost:8080/
